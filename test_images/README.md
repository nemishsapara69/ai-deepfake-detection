# Sample Test Images

This directory contains sample images for testing the deepfake detection system.

## Where to Get Test Images

### Real Face Images
1. **Unsplash**: https://unsplash.com/s/photos/portrait
2. **Pexels**: https://www.pexels.com/search/face/
3. **Kaggle Datasets**: Various face datasets
4. **Your own photos**: Upload clear face photos

### Fake/AI-Generated Face Images
1. **This Person Does Not Exist**: https://thispersondoesnotexist.com/
   - Generate random AI faces
   - Right-click and save image
   
2. **Generated Photos**: https://generated.photos/
   - AI-generated face library
   
3. **Midjourney/DALL-E**: 
   - Generate faces using text prompts
   
4. **StyleGAN**: 
   - Use StyleGAN models to generate faces

## Testing Tips

### Good Test Images
- Clear, well-lit faces
- Front-facing portraits
- Minimal occlusions
- High resolution (at least 512x512)
- Single face per image

### Poor Test Images (may fail)
- Multiple faces
- Side profiles
- Heavy filters/editing
- Low resolution
- Blurry or dark images
- No visible face

## Download Sample Dataset

For a quick test dataset, you can use:

1. **CelebA** (Real Faces)
   - Small subset: Download 50-100 images
   - Link: https://mmlab.ie.cuhk.edu.hk/projects/CelebA.html

2. **100K Faces** (Fake Faces)
   - Generated by StyleGAN
   - Link: https://generated.photos/datasets

## Testing Workflow

1. Download/collect 5-10 real face images
2. Download/generate 5-10 fake face images
3. Upload each to the application
4. Verify detection accuracy
5. Note any false positives/negatives

## Expected Behavior

- **Real faces**: Should show green indicator (Real)
- **AI-generated faces**: Should show red indicator (Fake)
- **Confidence**: Typically 70-95% for good quality images
- **Face detection**: Should automatically crop face region
